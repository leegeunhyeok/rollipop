## Features

All the features of React Native DevTools are available in Rollipop:

- [Console](https://reactnative.dev/docs/react-native-devtools#console)
- [Source & breakpoints](https://reactnative.dev/docs/react-native-devtools#sources--breakpoints)
- [Network](https://reactnative.dev/docs/react-native-devtools#network-since-083) (since `0.83`)
- [Performance](https://reactnative.dev/docs/react-native-devtools#performance-since-083) (since `0.83`)
- [Memory](https://reactnative.dev/docs/react-native-devtools#memory)
- [Built-in support for React DevTools](https://reactnative.dev/docs/react-native-devtools#react-devtools-features)

## Usage

To open DevTools, either:

- Select "Open DevTools" in the Dev Menu.
- Press `j` from the CLI.

<Callout>
Press `Shift + D` to enable automatic DevTools launch on app startup.
</Callout>
