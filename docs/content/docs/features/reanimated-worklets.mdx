---
title: Reanimated & Worklets
---

Rollipop ships with a built-in worklet transformation that enables the native worklet syntax used by [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/) (v4+). It runs as part of the SWC-based pass inside rolldown's [native transform pipeline](/docs/features/experimental#native-transform-pipeline).

<Callout type="warn">
  Worklet transformation is an experimental feature and only runs when
  `experimental.nativeTransformPipeline` is enabled. See [Experimental
  Features](/docs/features/experimental) for stability notes.
</Callout>

## Setup

Enable the native transform pipeline and turn worklets on under `experimental`:

```ts title="rollipop.config.ts"
import { defineConfig } from 'rollipop';

export default defineConfig({
  experimental: {
    nativeTransformPipeline: true,
    // Pass an empty object to opt in with defaults.
    worklets: {},
  },
});
```

The transformer automatically detects the installed `react-native-worklets` version (used as `__pluginVersion` in the output) and adjusts behavior based on the current `mode` (release builds skip debug info such as stack details, version, and source location).

## Configuration

`experimental.worklets` accepts the following fields. Pass `worklets: {}` to opt in with defaults; omit the field entirely to skip the transform.

| Property                     | Type       | Description                                                                                                       |
| ---------------------------- | ---------- | ----------------------------------------------------------------------------------------------------------------- |
| `globals`                    | `string[]` | Identifiers treated as globals — never captured into worklet closures.                                            |
| `strictGlobal`               | `boolean`  | When `true`, only the names listed in `globals` are considered safe.                                              |
| `omitNativeOnlyData`         | `boolean`  | Omit native-only data (`init_data`) from the output. Useful for web builds.                                       |
| `disableSourceMaps`          | `boolean`  | Disable source map generation for worklets.                                                                       |
| `relativeSourceLocation`     | `boolean`  | Use paths relative to `cwd` for source locations.                                                                 |
| `disableWorkletClasses`      | `boolean`  | Disable Worklet Classes support.                                                                                  |
| `disableInlineStylesWarning` | `boolean`  | Suppress the inline-shared-values warning.                                                                        |
| `bundleMode`                 | `boolean`  | Enable Bundle Mode.                                                                                               |
| `isRelease`                  | `boolean`  | Release builds skip debug info such as stack details, version, and location. Defaults to `mode === 'production'`. |
| `pluginVersion`              | `string`   | Version string emitted as `__pluginVersion`. Defaults to the installed `react-native-worklets` version.           |

## How It Works

The worklet transform converts functions marked with the `'worklet'` directive into a form that can be executed on the UI thread. Unlike Babel-based alternatives, the transform runs as a native SWC visitor inside rolldown's Rust core, alongside the rest of the [native transform pipeline](/docs/features/experimental#native-transform-pipeline) — avoiding the JS-side Babel round trip.

- **SWC-based**: Native performance without Babel overhead
- **Auto-detection**: Reads the `react-native-worklets` version from your project
- **Mode-aware**: Adjusts behavior based on `development` or `production` mode

## Requirements

- `experimental.nativeTransformPipeline` must be set to `true`
- `react-native-worklets` must be installed in your project
- Compatible with React Native Reanimated 4+
